# 🌐 Nest Web

Este repositorio contiene la implementación del sitio web de soporte legal y administrativo para la aplicación móvil **Nest**.

## 📄 Propósito

Para cumplir con las normativas de las tiendas de aplicaciones (Google Play Store y Apple App Store), este sitio proporciona el acceso público a la documentación legal necesaria para los usuarios de la aplicación:

- **Política de Privacidad:** Detalle del tratamiento de datos personales y seguridad.
- **Eliminación de Cuenta:** Flujo transparente y sencillo para que los usuarios soliciten la eliminación definitiva de sus datos y cuenta de la aplicación.

## 🛠️ Tecnologías

- HTML5 / CSS3
- Despliegue optimizado para acceso rápido.
